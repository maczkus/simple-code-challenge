# Simple Code Challenge

## Quick start

Clone the repository
```
git clone git@github.com:tulik/simple-code-challenge.git
```

Install dependencies

```
composer install
```

Test code:

```
compooser code-check
```

Run as web service:

```
php -S 127.0.0.1:8000 -t public
```
